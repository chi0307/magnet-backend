## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

## create API

[CRUD generator](https://docs.nestjs.com/recipes/crud-generator)

```bash
$ nest g resource XXX
```

## generate migration

```bash
$ FILE=XXX pnpm run migration:generate
```
